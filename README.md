# 📊 Unemployment Analysis using Python

This project focuses on analyzing unemployment trends using Python and data visualization techniques.

---

## 🚀 Features

- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Visualization of unemployment trends  
- Regional and temporal insights  
- Interactive Google Colab notebook  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn
